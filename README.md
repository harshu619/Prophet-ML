# Banking-Management-System
Console-based Banking Management System using C++ and OOP concepts with transaction management and account operations.
## Features

- Create Account
- Deposit Money
- Withdraw Money
- Balance Check
- Transaction History
- PIN Verification
- Account Search
- Menu Driven Program

## Concepts Used

- Classes and Objects
- Encapsulation
- Vectors
- File Handling (if added)
- Functions
- Loops and Switch Case
- OOP Principles

## Technologies Used

- C++
- VS Code
- Clang/G++

## How to Run

### Compile

```bash
clang++ main.cpp -o main
```

### Run

```bash
./main
```
